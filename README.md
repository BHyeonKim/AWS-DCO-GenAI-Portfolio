# AWS-DCO-GenAI-Portfolio

## 1. 과정명

AWS DCO 인턴십 직무 이해를 위한 생성형 AI 활용 및 포트폴리오 제작 과정

## 2. 포트폴리오 목적

이 저장소는 데이터센터 운영(DCO) 직무를 이해하기 위해, 생성형 AI를 학습 도구로 활용하며 진행한 실습 과정과 산출물을 정리한 학습형 포트폴리오입니다. 각 차시별로 DCO 직무 용어, 협업 역량, SOP 분석, 로그 분석, 장애 대응 문서 작성 등을 연습한 결과를 폴더별로 담고 있습니다.

## 3. 교육용 샘플 데이터 사용 안내

본 포트폴리오에 포함된 모든 예시, 로그, 장비명, SOP는 실제 AWS 내부 자료가 아니라 수업을 위해 만들어진 **교육용 샘플 데이터**입니다. 실습 자료에는 `SAMPLE`, `DEMO`, `EDU` 등 접두어가 붙은 이름을 사용해 실제 정보와 구분하고 있으며, 실제 장비 점검이나 실제 네트워크 진단을 수행한 결과가 아닙니다.

## 4. 각 폴더의 역할

| 폴더 | 역할 |
| --- | --- |
| `01_dco_glossary` | DCO 직무 핵심 용어와 인프라 용어를 정리한 용어집 |
| `02_dco_profile` | 나의 협업 강점과 보완점을 정리한 협업 유형 프로필 |
| `03_sop_analysis` | 교육용 샘플 SOP를 목적/절차/주의사항/확인항목으로 분석하고 체크리스트로 변환한 산출물 |
| `04_environment_setup` | 실습에 필요한 개발 환경(도구 설치, 설정 등)을 구성한 기록 |
| `05_antigravity_cli` | Antigravity CLI 실습 관련 산출물 |
| `06_cli_file_automation` | CLI를 활용한 파일 처리 자동화 실습 산출물 |
| `07_log_analysis_script` | 교육용 샘플 로그를 분석하는 스크립트 실습 산출물 |
| `08_incident_analysis` | 교육용 장애 시나리오를 분석한 실습 산출물 |
| `09_incident_report` | 교육용 장애 대응 보고서(Incident Report) 작성 실습 산출물 |
| `10_stitch_dco_dashboard` | Stitch를 활용한 DCO 대시보드 제작 실습 산출물 |
| `11_exam_prep` | 지필시험 대비 핵심 용어 및 개념 정리 자료 |

*(폴더별 역할은 폴더명을 기준으로 한 초안이며, 실제 실습 진행에 맞게 내용을 조정해 주세요.)*

## 5. 보안상 포함하지 않는 정보

이 포트폴리오에는 아래와 같은 정보를 포함하지 않습니다.

- 실제 AWS 내부 절차, 내부 SOP, 비공개 업무 지침
- 실제 회사명, 실제 고객명, 실제 고객 시스템 정보
- 실제 장비명, 장비 시리얼, 자산 태그, 실제 IP·도메인·네트워크 구성 정보
- 실제 티켓 번호, 실제 장애 기록
- 계정정보, 비밀번호, API 키, 토큰 등 인증 정보
- 개인정보(전화번호, 주소, 주민등록번호, 개인 이메일 전체 등)

## 6. 앞으로 추가할 산출물

- 각 차시별 실습 완료 후 해당 폴더에 학습 산출물(정의서, 체크리스트, 스크립트, 보고서 등) 추가
- 지필시험 대비를 위한 핵심 용어 및 프롬프트 구조 정리(`11_exam_prep`)
- 실습 진행 중 갱신되는 내용을 반영해 이 README도 함께 업데이트
